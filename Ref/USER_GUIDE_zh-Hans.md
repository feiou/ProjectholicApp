# Projectholic 完整用户手册（简体中文）

本手册为完整版本，面向自学用户。
术语统一使用应用界面中的中文显示名（参考 `Projectholic/zh-Hans.lproj/Localizable.strings`）。

## 1. 产品总览

Projectholic 是一个把项目管理与执行排程结合起来的系统，核心模块包括：
- `项目`
- `日历`
- `甘特图`（iPad/Mac 侧重）
- `跟进`
- `数据`
- `设置`

## 2. 完整使用路径是什么

常见完整闭环如下：
1. 在 `项目` 新建项目
2. 规划阶段/交付物
3. 按 `任务类型` 添加任务（`一般` / `多日` / `重复`）
4. 在 `日历` / `甘特图` 排程
5. 用 `进入专注` 执行
6. 用 `每日摘要` 与 `日终回顾` 做复盘
7. 在 `数据` 查看趋势

## 3. 首次启动

首次启动会进入引导创建流程，通常包括：
1. 介绍
2. 项目名称与颜色
3. 目标与方向
4. 交付物/阶段
5. 起始任务
6. 可选配置（截止日期、工时、周末策略）

建议完成引导后立刻做三件事：
1. 进入 `项目`
2. 新建三种类型任务
3. 到 `日历` 完成首次排程

## 5. 项目页（`项目`）详解

`项目` 是项目库和创建入口。

顶部常用控件：
- 排序（`chevron.up.chevron.down`）
- 新建（`plus`）

新增方式：
- `引导式（PMBOK）`
- `快速添加`

搜索与筛选：
- 搜索框：`搜索项目...`
- 标签筛选：`按标签筛选`

项目卡片长按/右键菜单：
- `收藏项目` / `取消收藏`
- `标签`
- `编辑项目`
- `复制项目`
- 删除

复制选项：
- `复制项目和任务`
- `仅复制项目`

订阅限制提示：
- 免费层通常可创建首个项目。
- 继续创建可能触发 Pro 限制。

## 6. 项目详情结构（`计划` / `时间线`）

项目详情通常有两种工作视图：
- `计划`
- `时间线`

`计划` 侧重结构与内容：
- 阶段
- 任务
- 子任务
- 项目目标/说明

`时间线` 侧重顺序与排程：
- 任务先后
- 时间安排
- 进度状态

常见分区：
- 阶段分区
- `独立任务`（未归类阶段）
- `未安排` 任务区

## 7. 阶段（交付物）使用规则

阶段用于把项目拆解成清晰里程碑。

可执行动作：
- 新增阶段
- 编辑阶段
- 阶段完成切换
- 阶段重排
- 删除阶段

删除阶段时通常会出现选择：
- 仅删除阶段，保留任务
- 连同任务一起删除

## 8. 任务创建与编辑基础

任务编辑可设置：
- 标题
- 备注
- 所属项目
- 计划时间
- 时长
- 截止时间
- 子任务
- 提醒与日历同步覆盖项
- `任务类型`

常见动作按钮：
- `编辑任务`
- `复制任务`
- `删除任务`

完成状态图标：
- `circle`：未完成
- `checkmark.circle.fill`：已完成

## 9. `任务类型` 核心概念

编辑器中的字段名：`任务类型`。

可选显示值：
- `一般`
- `多日`
- `重复`

不同类型会改变排程方式与图标提示。

## 10. `一般` 任务

适用于一次性、常规任务。

特点：
- 一般表现为单块任务
- 支持专注、复制、删除、完成切换

## 11. `多日` 任务

适用于总时长较长、需要跨天拆分的任务。

特点：
- 可自动分配到多天
- 支持按块完成
- 可在自动分配基础上手动微调

常见自动参数：
- 开始日期
- 每日最大分钟
- 最小块时长
- 最大块时长
- 是否跳过周末

跨天提示图标：
- 向后箭头：从前一天延续
- 向前箭头：延续到下一天

## 12. `重复` 任务

适用于固定节律任务。

`重复设置` 中常见术语：
- `每`
- `结束条件`
- `永不`
- `在日期`
- `次数`

可按天/周/月/年配置规则。

## 13. `日历` 执行指南

`日历` 是日常执行主场景。

你可以：
- 打开任务并编辑
- 快速勾选完成
- 查看当天排程密度
- 从顶部菜单进入：
  - `每日摘要`
  - `日终回顾`

任务块交互：
- 单击：打开详情/编辑
- 双击：快速进入专注（自动开始路径）
- 长按：打开任务菜单

## 14. `甘特图` 规划指南

`甘特图` 更适合跨项目时间规划。

可执行操作：
- 多项目时间线浏览
- 缩放时间粒度
- 按项目/标签筛选
- 在时间范围内跳转
- 打开任务详情

工具栏常见按钮：
- 筛选（`line.3.horizontal.decrease.circle`）
- 缩小（`minus.magnifyingglass`）
- 放大（`plus.magnifyingglass`）

## 15. iPad/Mac 的月历 `日历`

该视图适合中观排期。

能力：
- 按天查看任务密度
- 打开当天侧边面板
- 项目与标签筛选

筛选面板按钮：
- `清除`
- `完成`

## 16. 专注流程（`进入专注`）

专注计时与任务绑定。

入口：
- 任务菜单 `进入专注`
- 任务块双击快捷进入

会话生命周期：
- 开始
- 暂停
- 继续
- 结束

附加行为：
- 支持最小化
- 可从悬浮状态恢复
- 任务层保存专注历史

## 17. 跟进模块（`跟进`）

`跟进` 用于查看委派/共享任务。

可执行：
- 查看共享任务
- 按人/按日期分组
- 打开任务详情
- 重新指派
- 取消共享
- 标记完成

常见图标：
- `person.2.fill`
- `person.2.arrow.trianglehead.counterclockwise`
- `xmark.circle`

## 18. 数据模块（`数据`）

`数据` 用于趋势与绩效观察。

入口通常包括：
- 月视角
- 季视角
- 年视角
- 项目级明细

建议使用方式：
1. 看是否按计划推进
2. 看哪个项目偏离较大
3. 看整体完成节奏

## 22. 按钮与图标速查（用户提问直答）

当用户问“这个按钮干嘛用”：
- `xmark`：关闭当前页面/弹窗
- `chevron.right`：进入详情
- `ellipsis`：更多操作
- `square.and.pencil`：编辑（如 `编辑任务`）
- `doc.on.doc`：复制（任务或项目）
- `trash`：删除
- `plus` 系列：新增
- `checkmark.circle.fill` / `circle`：完成状态切换

- `line.3.horizontal.decrease.circle`：筛选
- `sparkles`：引导/亮点/升级相关入口
- `lock.fill`：功能受限或需 Pro

## 23. 图标图例（按区域）

## 导航
- `folder.fill` => `项目`
- `calendar` => `日历`
- `person.2.fill` => `跟进`
- `chart.xyaxis.line` => `数据`
- `gear` => `设置`
- `chart.bar.yaxis` => `甘特图`

## 任务类型提示
- `repeat` / `repeat.circle.fill` => `重复`
- `repeat.1` / `repeat.1.circle.fill` => 首次重复实例
- `arrow.backward(.circle.fill)` => 多日任务从前一天延续
- `arrow.forward(.circle.fill)` => 多日任务延续到下一天

## 完成状态
- `circle` => 未完成
- `checkmark.circle.fill` => 已完成
- `square` => 子任务/块未完成
- `checkmark.square.fill` => 子任务/块已完成

## 云同步
- 带锁云图标 => 同步需 Pro
- `icloud.slash` => 登录或账户状态异常
- 旋转云箭头 => 同步中
- 勾选云/勾选圆 => 同步成功
- 感叹号/叉号云 => 同步异常

## 24. 用户日常使用范式

## 范式 A：每日执行
1. 打开 `日历`
2. 看今日任务块
3. 对关键任务执行 `进入专注`
4. 即时勾选完成
5. 用 `每日摘要` 收尾

## 范式 B：每周规划
1. 打开 `项目`
2. 在 `计划` 调整结构
3. 用 `甘特图` 或月视图 `日历` 重排时间
4. 到 `数据` 查看效果

## 范式 C：委派管理
1. 打开 `跟进`
2. 按人/日期查看
3. 重新指派或取消共享
4. 确认完成情况

## 25. 新手推荐学习顺序

1. 先会 `项目` 新建与排序
2. 理解三种 `任务类型`
3. 学会 `日历` 中的任务块操作
4. 学会 `进入专注`
5. 学会 `跟进` 的委派闭环
6. 学会在 `数据` 看趋势
7. 完成 `设置` 里的 `工作时间` 与 `默认标签页`

## 26. 问题排查手册

## “我找不到任务”
请检查：
- 日期范围
- 筛选条件
- `未安排` 区域
- `独立任务` 分区

## “为什么不能继续新建项目”
通常是免费层级限制。

## “同步没更新”
在 `iCloud 状态` 里先看状态，再点 `刷新`。

## “多日任务排程怪怪的”
打开任务确认：
- `任务类型` 是 `多日`
- 自动分配参数是否合理

## “重复任务行为不符合预期”
进入 `重复设置`，检查：
- `每`
- `结束条件`
- 规则频率与终止条件

## 27. 客服答复模板（针对按钮/图标问题）

当用户问某个按钮：
1. 先确认所在页面（`项目`/`日历`/`甘特图`/`跟进`/`数据`/`设置`）
2. 确认图标外形与邻近文案
3. 用本手册第 22/23 节映射
4. 若是删除类操作，补充影响与确认提示

## 28. 术语表

- 项目：顶层容器（`项目`）
- 阶段：项目内交付物分段
- `一般`：一次性任务类型
- `多日`：长任务跨天分配类型
- `重复`：规则循环任务类型
- `未安排`：尚未进入时间槽的任务
- 专注会话：由 `进入专注` 启动的计时记录
- `跟进`：委派/共享任务追踪页面
